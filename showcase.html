GEMINI RING
AUTOMATION

An autonomous bridge between local jewelry design and browser-based AI generation using Pinchtab.

The Vision

This project automates the complex task of gemstone replacement in ring designs. By leveraging the Google Gemini web interface without the need for expensive API keys, it provides a cost-effective, high-fidelity solution for jewelry product photography.

System Architecture

The core engine pinchtab_automation.py orchestrates the entire flow: capturing user input, processing images with PIL, and communicating with the Gemini web app via pinchtab_gemini.py.
